# Atividade de Aprendizado de Máquina: Algoritmo KNN 

Este repositório contém a resolução da atividade prática de Aprendizado de Máquina, focada na implementação, calibração e análise do algoritmo **KNN (K-Nearest Neighbors)** utilizando a biblioteca `scikit-learn` no dataset *Breast Cancer*.

---
## 🛠️ Tecnologias e Bibliotecas Utilizadas
* **Linguagem:** Python
* **Ambiente:** Jupyter Notebook (VS Code)
* **Principais Bibliotecas:** `pandas`, `numpy`, `matplotlib`, `seaborn`, `scikit-learn`

---

## 🚀 Como Executar o Notebook

1. **Clone o repositório:**
   ```bash
   git clone [https://github.com/seu-usuario/nome-do-repositorio.git](https://github.com/seu-usuario/nome-do-repositorio.git)
    ```
2. **Instale as dependências necessárias:**
   ```bash
   pip install pandas numpy matplotlib seaborn scikit-learn
    ```
2. **Execute o arquivo:**
 Abra o arquivo .ipynb em seu ambiente Jupyter local, certifique-se de selecionar o kernel Python correto com as dependências instaladas e execute as células sequencialmente.
